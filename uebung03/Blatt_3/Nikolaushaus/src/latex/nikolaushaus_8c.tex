Zeichnet Nikolaushaus in eigenem Fenster. In diesem Programm wird mithilfe von S\-D\-L ein rotes Nikolaushaus in einem Fenster gezeichnet. Der S\-D\-L-\/\-Rendere übergibt dabei die Aufgabe des zeichnens an die Grafikkarte.
